# BoogieBrainsMSHub

An e-commerce website with:
- Admin panel to add products
- Wallet system in Naira
- Flutterwave/Paystack integration
- Customer login/register
- Order history

## How to run
1. Clone repo: `git clone https://github.com/YOURUSERNAME/boogiebrainsmshub`
2. Backend: `cd backend && npm install && npm start`
3. Frontend: `cd frontend && npm install && npm start`
